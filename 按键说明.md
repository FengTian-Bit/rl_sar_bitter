# 键盘控制说明  
---

## 1. 使用前准备
- 确保终端窗口处于**激活状态**，再按键盘。  
- 所有按键**区分大小写**，请关闭 CapsLock。  
- 每次按键立即生效，无需回车。
---

## 2. 模式切换（状态机FSM）
| 按键 | 功能说明 |
|------|----------|
| 0 | 进入 **起立** 模式（POS_GETUP） |
| p | 进入 **强化学习 Locomotion** 模式（RL_LOCOMOTION） |
| o | 进入 **轮式** 模式（WHEEL） |
| n | 进入 **强化学习导航** 模式（RL_NAVIGATION） |
| 1 | 进入 **趴下** 模式（POS_GETDOWN） |
| r | **重置仿真**（RESET_SIMULATION） |
| 回车 | **暂停/继续仿真**（TOGGLE_SIMULATION） |
| 2 | 进入测试模式 STATE_TEST2 |

## 3. 实时运动微调（仅在支持的模式下生效）
| 按键 | 物理量 | 步长 | 说明 |
|------|--------|------|------|
| w | x 方向线速度 | +0.1 m/s | 前进 |
| s | x 方向线速度 | -0.1 m/s | 后退 |
| j | y 方向线速度 | +0.1 m/s | 左平移 |
| l | y 方向线速度 | -0.1 m/s | 右平移 |
| a | yaw 角速度 | +0.1 rad/s | 左转 |
| d | yaw 角速度 | -0.1 rad/s | 右转 |

## 4. 紧急清零
| 按键 | 功能 |
|------|------|
| 空格 | 立即把 **x、y、yaw** 三项速度全部置零（急停） |

## 5. 未知按键
若误按其他键，终端会打印：  
`Unknown key pressed`  
可据此确认按键是否被识别。

## 6. 常用操作流程示例
1. 启动程序后，先按 **0** 让机器人起立。  
2. 按 **p** 进入 RL 运动模式。  
3. 用 **w/a/s/d** 实时微调速度；需要急停时按**空格**。  
4. 想重新开始，先按 **r** 重置仿真，再按 **0** 起立。  
5. 调试过程中可用**回车**暂停/继续仿真。

```markdown
# ROS 1 遥控手柄使用说明
```
> 使用**Xbox 2.4 GHz 无线接收器** + **ROS Noetic** 

## 1 硬件连接
1. 将 2.4 GHz 接收器插入电脑 USB 口  
2. **单击手柄 Logo 键** 开机  
   - 连接成功：**手柄震动 + 指示灯黄灯常亮**  
   - 若无震动，长按 Logo 键 3 s 重新配对
---

## 2 安装依赖
```bash
sudo apt update
sudo apt install ros-noetic-joy 
```

## 3 模式切换（状态机）
| 按键组合 | 功能说明 |
|-----------|----------|
| **LT+Y** | 进入 **起立** 模式（POS_GETUP） |
| **LT+A** | 进入 **趴下** 模式（POS_GETDOWN） |
| **LT+B** | 进入 **强化学习 Locomotion** 模式（RL_LOCOMOTION） |
| **LT+X** | 进入 **强化学习导航** 模式（RL_NAVIGATION） |
| **Back（中左）** | **重置仿真**（STATE_RESET_SIMULATION） |
| **Start（中右）** | **暂停/继续仿真**（STATE_TOGGLE_SIMULATION） |

> 摇杆映射：<br>
> • 左摇杆 ↑↓ **前后**（x 0.75 m/s）<br>
> • 左摇杆 ←→ **左右平移**（y 0.5 m/s）<br>
> • 右摇杆 ←→ **原地旋转**（yaw ±0.25 rad/s）
